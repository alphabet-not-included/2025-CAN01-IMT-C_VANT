# Controle de Potência do Sistema Propulsivo de um VANT

## Sobre o Projeto

Este repositório contém o código-fonte, diagramas de circuitos e outros recursos desenvolvidos como parte do Trabalho de Conclusão de Curso apresentado em 2025 no curso de Engenharia de Controle e Automação pelo Centro Universitário do Instituto Mauá de Tecnologia.

### Resumo

O projeto tem como finalidade desenvolver um controlador de potência para o propulsor de um VANT (Veículo Aéreo Não Tripulado), composto por um motor brushless de corrente contínua (BLDC), visando otimizar o desempenho energético da aeronave e minimizar penalidades durante a competição SAE Aerodesign brasileira.
